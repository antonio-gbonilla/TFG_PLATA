El proyecto PLATA consiste en el desarrollo de un lenguaje de programación específico orientado al control de vehículos autónomos agrícolas, junto con su intérprete implementado en Java utilizando ANTLR4. Este lenguaje permite definir el comportamiento de un tractor mediante instrucciones de alto nivel, que son procesadas y ejecutadas por un intérprete capaz de operar tanto en modo consola como conectado a un entorno de simulación.

El repositorio está organizado en dos partes principales: por un lado, el módulo Gradle_exteralController, que contiene toda la lógica del lenguaje, la gramática y el intérprete; y por otro, el módulo MundoTractor, que incluye el entorno de simulación en Webots con el modelo del tractor y su configuración. Ambas partes están diseñadas para trabajar conjuntamente mediante el uso de un controlador externo.

Para ejecutar el sistema en modo simulación, es necesario abrir previamente el mundo de Webots y configurar el robot con un controlador externo, de forma que el intérprete pueda conectarse y enviar las instrucciones generadas a partir del programa en lenguaje PLATA. De este modo, se consigue una integración completa entre el lenguaje desarrollado y la simulación del comportamiento del vehículo.
